What is MiC?

The Maths in Colors (MiC)s website is a template that you could use for a (small) academic conference. It’s got a page for registration, where you can embed a Google Form or some other way for participants to register. It’s got a program page to list the schedule for the conference. It’s got a directions page to add a bunch of instructions how to get to the conference. It’s got a page to put up the flyers and other promotional materials for folks to download.

You can use this template by cloning all the files from this GitHub repo here. This template was created from a website I made for MathConnections 2019, in case you want to see an example of this template being used in the wild.
